자주 묻는 질문 (FAQ)

아래 답변은 실제 이용 중 자주 들어오는 문의를 기준으로 정리했습니다. 추가 문의는 문의 페이지에서 접수해 주세요.

1. 결과가 실제 연봉을 정확히 예측하나요?

아니요. 본 서비스는 엔터테인먼트와 자기점검 목적의 도구입니다. 결과는 절대적인 예언이 아니라, 현재 상태를 돌아보는 참고 지표로 사용해야 합니다.

2. 사진은 서버에 저장되나요?

저장하지 않습니다. 얼굴 분석은 브라우저(사용자 기기)에서 처리되며, 운영자가 원본 사진을 수집하거나 보관하지 않습니다.

3. 같은 사람인데 결과가 달라지는 이유는?

조명, 촬영 각도, 표정, 거리, 해상도 차이로 입력 데이터가 달라지면 결과도 달라질 수 있습니다. 정면, 자연광, 무보정 사진으로 2~3회 반복해 공통 패턴을 확인하는 방식을 권장합니다.

4. 어떤 식으로 결과를 활용하면 좋나요?

결과를 역량, 시장, 협상 세 축으로 분해해 실행 계획으로 바꾸는 것이 좋습니다. 예를 들어 4주 단위로 포트폴리오 1개 완성, 성과 문서화 2건 작성처럼 측정 가능한 목표를 두세요.

5. 광고가 나오지 않거나 레이아웃이 깨져요.

광고 차단 확장 프로그램, 브라우저 프라이버시 설정, 캐시 문제로 발생할 수 있습니다. 시크릿 창 재확인, 다른 브라우저 테스트, 캐시 삭제 후 재시도해 주세요.

6. 콘텐츠는 어떻게 작성되나요?

블로그 글은 운영팀이 직접 작성하며, 동일 주제라도 실제 적용 가능한 체크리스트와 해석 기준을 포함하도록 편집합니다. 자동 복제·스크랩 콘텐츠는 운영 정책상 게시하지 않습니다.

7. 오탈자나 사실 오류를 발견하면?

문의 페이지 또는 이메일로 제보해 주세요. 확인 후 수정 이력을 남기고 재발 방지를 위해 내부 점검 항목에 반영합니다.
